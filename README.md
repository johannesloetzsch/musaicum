# musaicum

**Mosaic generation by algorithms for 2d bin packing**

## Run

``` shell
yarn install

yarn watch
```

[http://localhost:8700/](http://localhost:8700/)

## Clean

``` shell
yarn clean
```

## Release

``` shell
yarn release
```
